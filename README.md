# Prompt Engineering Portfolio

This repository documents my learning journey in Prompt Engineering through the "Prompt Engineering for ChatGPT" course offered by Vanderbilt University on Coursera.

## What I Learned

Throughout the course, I explored a wide range of prompt engineering concepts, patterns, and techniques for working effectively with Large Language Models (LLMs). Some of the key topics included:

* Persona Pattern
* Audience Persona Pattern
* Flipped Interaction Pattern
* Question Refinement Pattern
* Cognitive Verifier Pattern
* Outline Expansion Pattern
* Fact-Checking Techniques
* Few-Shot Prompting
* Prompt Design Best Practices
* Structured Prompt Development
* Prompt Evaluation and Improvement Strategies
* AI-Assisted Problem Solving Techniques
* Creative and Analytical Prompting Methods
* Human-AI Collaboration Approaches

In addition to these topics, I gained practical experience in designing, refining, and testing prompts to improve the quality, accuracy, and usefulness of AI-generated responses. The course also helped me develop a deeper understanding of how human creativity and effective prompting can unlock the full potential of Large Language Models.
